### Prerequisites

- [Node.js](https://nodejs.org/) (version 14 or newer recommended)  
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/mahianyuallan/personal-site-template.git

# Change directory
cd personal-site-template
```

# Install dependencies
```
npm install
# or
yarn install
```

# Start dev server
```
npm run dev
# or
yarn dev
```
